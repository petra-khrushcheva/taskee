# Taskee - таск менеджер

***
### Технологии
Python 3.9  
FastAPI 0.68.1  
SQLModel 0.0.4  
***
### Установка
- Клонируйте проект:
```
git clone <ссылка>
``` 
- Перейдите в директорию taskee:
```
cd taskee
``` 
- Cоздайте .env файл по образцу.
- Запустите Docker-compose:
```
docker-compose up -d
```
